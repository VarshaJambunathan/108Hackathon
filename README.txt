1. Run the web on a server.
2. Google Chrome is the most preferred browser. There is a possiblilty that a Cross Origin error might occur, so please add the CORS extension in Chrome.
3. Download the .apk on an Android phone. 
4.Ping a request from the phone.
5.The ERC Dashboard will be automatically updated with the new ping.
6.Copy the Latitude and Longitude from the 'Dashboard' to the 'Parse Lat-Lon' tab.
7.Paste the Lat-lon data and click on 'PARSE'
8.The shortest four distances with the addresses are displayes in the ascending order.
